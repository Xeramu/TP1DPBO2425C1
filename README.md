  1. Class TokoElektronik
     Atribut (private):
     - Nama Barang
     - Kode Barang
     - Stok Barang
     - Harga Barang
  
     Method (public):
     - constructor

     - destructor

     - Create
     - Read
     - Update
     - Delete
     - Search

<< Flowcode >>
  1. Melakukan perulangan do while, minta input user pilihan 1-6. Selama bkn 6, loop terus jalans
     
  2. Pilihan 1 (CREATE)
     - User diminta input nama, kode, stok, harag
     - Barang baru dibuat lalu di masukan kedalam vector
     - Print pesan berhasil
    
  3. Pilihan 2 (READ)
     - for loop smua barang yang ada dalem vector
     - print isinya
  
  4. Pilihan 3 (UPDATE)
     - user masukin kode yg mau diupdate
     - nyari kode pake fungsi nyari kode
     - klo ketemu, user diminta input data buat update
     - klo engga nemu, print pesan error dan balik ke look do while
  
  5. Pilihan 4 (DELETE)
     - user masukin kode barang yang mau diapus
     - nyari kode pake fungsi
     - klo emu, barang dengan kode tsb diapus
     - klo engga, print pesan error dan balik ke loop do while
       
  6. Pilihan 5 (SEARCH)
      - user masukin kode yang mau diacri
      - nyari kode pake fungsi
      - klo nemu, print barang dengan kode tsb
      - klo engga, print pesan error dan balik ke loop do while
        
  7. Pilihan 6 (EXIT)
      - print pesan dadah
      - exit program
